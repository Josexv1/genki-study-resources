Exercises are Stuck Loading

When using Genki Study Resources offline, sometimes you may run into an issue where exercises will endlessly load. This is usually related to the blocking of third-party cookies, which for whatever reason, blocks cookies and localStorage of locally used files giving you the following result.

Thankfully this can easily be fixed by adjusting some settings in your browser, unless of course you're using Mircosoft Edge, which you shouldn't, because that has a bug (or super-paranoid security) that doesn't let local files function at all even when all cookies are allowed.

Adjusting Browser Settings

To fix this issue, simply go to settings in your browser and search for cookie related settings, and look for an option that says "Block third-party cookies" or "Only block cross-site cookies" and either change it to "Allow all" or turn it off. This is a common issue with Chromium-based browsers, so it should be easy to fix by following this guide, but regardless, I'll list individual browser instructions for affected browsers below.

Google Chrome and Opera

Go to Settings > Privacy and security > Site Settings > Cookies and site data and turn Block third-party cookies off.

Brave

Go to Settings > Shields > Cookies and set it to Allow all cookies.

Microsoft Edge

Download either Brave, Opera, or Chrome.

Exercises are Still Stuck Loading

If you're still having trouble with the endlessly loading exercises, please open a new issue on Github, so that we can assist you.
